## 🚀 开发

```bash
# 配置
1. 一键安装 .vscode 目录中推荐的插件
2. node 版本 18.x 或 20+
3. pnpm 版本 8.x 或最新版

# 安装依赖
pnpm i

# 启动服务
pnpm dev
```

## 项目目录结构说明

```
vue3-data
├─ src
│  ├─ App.vue
│  ├─ api           --接口
│  ├─ assets        --静态资源目录
│  ├─ components    --公共组件
│  ├─ config        --配置
│  ├─ constants     --Key
│  ├─ directives    --权限及指令
│  ├─ hooks         --自定义hooks
│  ├─ icons         --svgIcons
│  ├─ lang          --多语言语种
│  ├─ layouts       --布局
│  ├─ messages      --多语言配置
│  ├─ plugins       --静态资源
│  ├─ router        --路由
│  ├─ store         --状态
│  ├─ styles        --公共样式
│  ├─ utils         --公共函数
│  ├─ views         --页面
```
